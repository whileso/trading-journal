## 核心纪律

1. 叠BUFF ≥ 3个才开单
2. 盈亏比 ≥ 1.5:1 才开单
3. 每单风险 ≤ 总资金的 1-2%
4. 每单必填交易记录
5. 大级别定方向，小级别找入场
